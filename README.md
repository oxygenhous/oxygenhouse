# Oxygen House — iGAS

نظام تقارير صيانة الغازات الطبية (iGAS): تطبيق ويب لملء وإدارة تقارير الصيانة الدورية لشبكة الغازات الطبية بدل التقرير الورقي، مع طباعة تطابق شكل الـ PDF الأصلي.

This project is linked to the private repository: https://github.com/oxygenhous/oxygenhouse

## Tech Stack

- **Frontend + Backend:** Next.js (App Router) + TypeScript + Tailwind CSS
- **Database + Auth:** Supabase (Postgres + Supabase Auth)
- **Hosting:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Copy `.env.local.example` to `.env.local` and fill in your Supabase project URL and anon key before running.
